Android Mortgage Calculator App (PROJECT)

Android Studio & Java • Fall 2020

- Created a simple android app that calculates mortgage rates for the provided terms and interest rates and displays them on the screen. It was built with Android studio in Java.
- This is my first app created with Android studio. it is a very simple app that utilizes Layouts, Intents, and Widgets to create a somewhat elegant UI.
- Utilized: Java, Android Studio

Java RMI Debt Consolidation Tool (PROJECT)

Java & Java RMI • Fall 2020

- Utilized JAVA remote method invocation to create a client-server based debt consolidation tool. Designed using Maven and IntelliJ
- A simple command line based tool that verifies Canadian SIN numbers and other services to create a report of debts and assets for students. The purpose of this project was to demonstrate knowledge of JAVA RMI tools.
- Utilized: Java, RMI

Heroku Cloud Based NodeJS App (PROJECT)

Heroku & NodeJS • Winter 2021

- Leveraged knowledge to create an app called “Sticky-Notes” which allows people to leave memories in the form of messages from anywhere in the world, and then later view them on a map with pins. it also sends an email alert when the message is initially sent from the user with the longitude and latitude of the location of the message.
- This app has a very basic UI, but a great idea. I think it's great for people who don't have memory issues.
- Utilized: NodeJS, SendGrid Twilio EmailAPI, GoogleMaps API, PostGresQL, Heroku

Raspberry Pi Based IoT Fire Detection Device (TEAM PROJECT)

Raspberry Pi 4 & Python • Winter 2021

- Leveraged knowledge to createRaspberry Pi IoT based fire detection device which uses multiple sensors onboard to detect environmental conditions to predict the possibility of forest fires using algorithms developed.
- Sensors would send data to Cloud platform and then retrieved by a Python software which then inputted data into our program to predict the possibility of fires using our algorithms developed.
- Utilized: Python, ThingSpeak IoT Cloud Platform, Linux, Raspberry Pi, GPiO Hardware, Agile, Electric Circuits
